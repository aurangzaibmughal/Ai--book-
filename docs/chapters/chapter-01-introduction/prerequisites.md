# Prerequisites

Before starting this chapter, you should have:

## Required Knowledge

### Programming
- **Python 3.x**: Basic understanding of Python syntax, functions, and object-oriented programming
- **Command Line**: Familiarity with terminal/command prompt operations
- **Git**: Basic version control concepts (clone, commit, push)

### Mathematics
- **Linear Algebra**: Basic understanding of vectors and matrices
- **Geometry**: 3D coordinate systems and transformations

### Computing
- **Operating System**: Ubuntu 22.04 LTS (recommended) or Windows with WSL2
- **Text Editor/IDE**: VS Code, PyCharm, or similar

## Optional but Helpful

- Basic understanding of robotics concepts
- Familiarity with ROS (Robot Operating System)
- Experience with simulation environments
- Knowledge of machine learning fundamentals

## Hardware/Software Requirements

### Minimum (Cloud-Based)
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Stable internet connection (10 Mbps+)
- AWS/GCP/Azure account for cloud instances

### Recommended (Local Development)
- Ubuntu 22.04 LTS
- 16GB RAM
- 50GB free disk space
- Mid-range GPU (optional for simulation)

### Premium (Physical Robotics)
- All recommended requirements
- Physical robot platform (optional)
- Edge computing device (Jetson Nano/Orin)

## Pre-Chapter Setup

1. **Install Python 3.11+**
   ```bash
   python3 --version  # Should be 3.11 or higher
   ```

2. **Verify Git Installation**
   ```bash
   git --version
   ```

3. **Create Working Directory**
   ```bash
   mkdir ~/robotics-learning
   cd ~/robotics-learning
   ```

## No Prerequisites?

If you're missing any prerequisites, don't worry! We'll guide you through the basics as we go. The hands-on labs include detailed setup instructions.
