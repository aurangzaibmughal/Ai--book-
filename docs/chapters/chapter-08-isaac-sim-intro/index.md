# Chapter 8: NVIDIA Isaac Sim Introduction

## Overview

Introduction to NVIDIA Isaac Sim, the GPU-accelerated robotics simulator built on Omniverse.

## What You'll Learn

- Isaac Sim architecture and capabilities
- Omniverse platform basics
- USD (Universal Scene Description)
- GPU-accelerated physics (PhysX)
- Photorealistic rendering with RTX
- ROS 2 integration with Isaac Sim

## Prerequisites

- Completed Chapter 7: Unity Simulation
- NVIDIA GPU (RTX series recommended)
- Understanding of parallel computing concepts

4-5 hours
