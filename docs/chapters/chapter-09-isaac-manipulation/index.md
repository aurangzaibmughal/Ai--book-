# Chapter 9: Robot Manipulation with Isaac Sim

## Overview

Master robot manipulation tasks using NVIDIA Isaac Sim, including grasping, pick-and-place, and dexterous manipulation.

## What You'll Learn

- Manipulation task setup in Isaac Sim
- Motion planning with Isaac Sim
- Grasp synthesis and planning
- Force/torque control
- Object manipulation strategies
- Sim-to-real transfer for manipulation

## Prerequisites

- Completed Chapter 8: Isaac Sim Introduction
- Understanding of kinematics and dynamics
- Familiarity with manipulation concepts

5-6 hours
