# Chapter 4: URDF & Robot Modeling

## Overview

Master the Unified Robot Description Format (URDF) to create accurate robot models for simulation and control.

## What You'll Learn

- URDF syntax and structure
- Links, joints, and kinematic chains
- Visual and collision geometries
- Inertial properties
- Xacro for modular robot descriptions
- Robot visualization in RViz2

## Prerequisites

- Completed Chapter 3: ROS 2 Workspace
- Basic understanding of 3D geometry
- XML/YAML familiarity

## Estimated Time

3-4 hours
