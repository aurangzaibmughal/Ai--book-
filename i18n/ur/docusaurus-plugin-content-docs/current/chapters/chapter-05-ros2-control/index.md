# باب 5: ROS 2 کنٹرول فریم ورک

## جائزہ

روبوٹ ہارڈویئر انٹرفیسز اور کنٹرولرز کے انتظام کے لیے ROS 2 کنٹرول فریم ورک کو دریافت کریں۔

## آپ کیا سیکھیں گے

- ros2_control آرکیٹیکچر
- ہارڈویئر انٹرفیسز اور تجریدات
- کنٹرولر مینیجر
- عام کنٹرولرز (joint trajectory، differential drive)
- ریئل ٹائم کنٹرول کے تحفظات
- ہارڈویئر انضمام

- باب 4 مکمل: URDF اور روبوٹ ماڈلنگ
- کنٹرول تھیوری کی بنیادی باتوں کی سمجھ
- C++ کا علم (مددگار لیکن ضروری نہیں)
